## 1. Regras de Formatação

1. **Markdown padrão**  
   - Títulos: `#`, `##`, `###`  
   - Ênfase: `*itálico*`, `**negrito**`  
   - Listas: `-` ou `1.`  

2. **Parágrafos lógicos**  
   - Separe ideias em parágrafos curtos.  
   - Se um parágrafo ficar muito longo, quebre-o em sentenças ou listas.

3. **Blocos de código**  
   - Sempre use três crases (\`\`\`) antes e depois.  
   - Dentro do bloco, cada item ou etapa em seu próprio parágrafo.  
   - **Exemplo:**
     ```python
     def exemplo():
         # Etapa 1: inicialização
         x = 1

         # Etapa 2: processamento
         y = x + 2

         # Etapa 3: retorno
         return y
     ```

4. **Gerenciamento de limite de caracteres**  
   - **Modo Padrão:** até 96 caracteres por linha.  
   - **Modo Pós-Fusão:** sem limite; use títulos, listas e blocos de código para clareza.  
   - Se ultrapassar 96 caracteres em Modo Padrão, mencione que detalhes completos virão após fusão.
